1.1
O Problema
Por que precisamos persistir dados?
🤔 E se fecharmos o programa agora?

Todos os cadastros somem! Tudo que estava na lista de dicionários era guardado na memória RAM — e a RAM é apagada quando o programa fecha. Precisamos "guardar" essas informações de forma permanente.

🐍 Seu programa Python
lista_alunos = [{"nome": "Ana"}, ...]
↕️ dados na memória RAM
❌ Fecha o terminal
POOF! Tudo sumiu! 💨

É aí que entra o TinyDB — um mini banco de dados que funciona direto no Python, sem precisar aprender outra linguagem!

1.2
O que é TinyDB?
Um banco de dados que cabe no bolso
📦 TinyDB = Tiny Database

Banco de dados pequeno. É uma biblioteca Python que salva seus dicionários automaticamente num arquivo. Sem SQL, sem configuração, sem complicação! Funciona com o que você já conhece: dicionários e listas.

Analogia: Do caderno à gaveta

Se a lista de dicionários era um caderno que você apagava ao fechar, o TinyDB é um caderno que fica salvo na gaveta. Quando você abre de novo, tudo continua lá! 📒

🐍 Seu programa Python
db.insert({"nome": "Ana", "idade": 22})
↕️ TinyDB cuida de tudo
💾 TinyDB
Salva automaticamente no arquivo
↕️ arquivo JSON no disco
📄 alunos.json
Dados persistem mesmo após fechar! ✅

Terminalterminal

O TinyDB é instalado com um único comando. Não precisa configurar nada! Sem servidor, sem senha, sem configuração de porta. É só importar e usar.

🔍 O que aconteceu no arquivo? (Clique para ver) ▼

O TinyDB criou automaticamente o arquivo alunos.json na mesma pasta do seu script. Se você abri-lo, verá algo assim:

alunos.jsonJSON

  "_default": {
    "1": {"nome": "Ana", "idade": 22, "curso": "Python"},
    "2": {"nome": "João", "idade": 25, "curso": "Java"}
  }
}
📁
O TinyDB usa um arquivo JSON por baixo dos panos. Cada registro recebe um ID automático ("1", "2", ...). A chave "_default" é a tabela padrão.

🔍 Entendendo o where()

O where funciona como um filtro: "me traga todos os registros onde o nome é Ana". É como perguntar ao banco: "Quem aqui se chama Ana?"

💡
O search() sempre retorna uma lista, mesmo que encontre apenas um resultado. Se não encontrar nada, retorna uma lista vazia [].

Resumo CRUD
As quatro operações fundamentais
🧠 O que é CRUD?

CRUD é a sigla para as 4 operações que todo banco de dados faz: Create (criar), Read (ler), Update (atualizar) e Delete (deletar). Com o TinyDB, cada operação é uma única linha de código!

🐍 CRUD com TinyDB
Operação Código TinyDB O que faz
CREATE db.insert({"nome": "Ana"}) Adiciona um novo registro
READ db.all() ou db.search(where(...)) Lista ou busca registros
UPDATE db.update({...}, where(...)) Altera um registro existente
DELETE db.remove(where(...)) Remove um registro

"Quatro operações. Quatro linhas. Sem SQL. Tudo Python." 🐍

Analogia: Gavetas de um armário

É como ter várias gavetas dentro do mesmo armário! O arquivo escola.json é o armário. Cada .table() é uma gaveta separada. A gaveta "alunos" não se mistura com a gaveta "professores". 🗄️

from tinydb import TinyDB, where

db = TinyDB("alunos.json")

def mostrar_menu():
    print("\n1 - Adicionar aluno")
    print("2 - Listar alunos")
    print("3 - Buscar por nome")
    print("0 - Sair")

def adicionar_aluno():
    nome = input("Nome: ")
    idade = int(input("Idade: "))
    curso = input("Curso: ")
    db.insert({"nome": nome, "idade": idade, "curso": curso})
    print("✅ Aluno cadastrado!")

def listar_alunos():
    alunos = db.all()
    if len(alunos) == 0:
        print("Nenhum aluno cadastrado.")
        return
    for aluno in alunos:
        print(f"- {aluno['nome']}, {aluno['idade']} anos, {aluno['curso']}")
    print(f"Total: {len(alunos)} aluno(s)")

def buscar_por_nome():
    nome = input("Nome para buscar: ")
    resultado = db.search(where("nome") == nome)
    if len(resultado) == 0:
        print("Nenhum aluno encontrado.")
    else:
        for aluno in resultado:
            print(f"- {aluno['nome']}, {aluno['idade']} anos, {aluno['curso']}")

while True:
    mostrar_menu()
    op = input("Escolha: ")

    if op == "1":
        adicionar_aluno()
    elif op == "2":
        listar_alunos()
    elif op == "3":
        buscar_por_nome()
    elif op == "0":
        print(f"Total de alunos no banco: {len(db.all())}")
        print("Até logo! 👋")
        break

with aba_venda:
    st.header("Registrar venda")
    produtos = estoque.all()
    if len(produtos) == 0:
        st.info("Nenhum produto em estoque para vender.")
    else:
        opcoes = ["Selecione um produto..."]
        mapa_produtos = {}
        for p in produtos:
            if p["qtd"] > 0:
                label = f"#{p['id']} - {p['nome']} (estoque: {p['qtd']})"
                opcoes.append(label)
                mapa_produtos[label] = p

        produto_escolhido = st.selectbox("Produto", opcoes, key="venda_prod")

        if produto_escolhido != "Selecione um produto...":
            produto = mapa_produtos[produto_escolhido]
            qtd_venda = st.number_input(
                "Quantidade", min_value=1,
                max_value=produto["qtd"], step=1, key="venda_qtd"
            )
            cliente = st.text_input("Nome do cliente", key="venda_cliente")
            total = produto["preco"] * qtd_venda
            st.info(f"💰 Total da venda: **R$ {total:.2f}**")

            if st.button("Confirmar Venda", type="primary"):
                if cliente.strip() == "":
                    st.error("Digite o nome do cliente!")
                else:
                    vendas.insert({
                        "id_produto": produto["id"],
                        "produto_nome": produto["nome"],
                        "qtd_venda": int(qtd_venda),
                        "preco_unit": produto["preco"],
                        "total": total,
                        "cliente": cliente.strip()
                    })
                    estoque.update(
                        {"qtd": produto["qtd"] - int(qtd_venda)},
                        where("id") == produto["id"]
                    )
                    st.success(f"✅ Venda registrada! Total: R$ {total:.2f}")
                    st.balloons()
                    st.rerun()

        if len(opcoes) == 1:
            st.warning("Todos os produtos estão com estoque zerado!")

Linha 15: O selectbox mostra a lista de produtos. O usuário clica e escolhe — sem digitar nada!
Linhas 19-22: max_value=produto["qtd"] impede que o usuário venda mais do que tem em estoque.
Linhas 31-33: Registramos a venda com vendas.insert() — salvando quem comprou, o que, quanto e o total.
